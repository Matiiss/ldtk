This guided tour explains how to create and use Biome enums.
  1. Open the Enums panel
  2. Create an Enum which will contain your possible Biome values (eg. Forest, Desert, Snow etc)
  3. Open the Level Settings panel
  4. Edit the Level custom fields
  5. Add an Enum field using your new Biome enum (NOTE: arrays are supported, you may have multiple Biome enums per level)
  6. Open the Layers panel
  7. Pick the field you created as the layer "Biome enum"
  8. Close the Layers panel
  9. Open this layer rules
  10. You may now assign a Biome condition to each of your layer group of rules. For example, if you have a group of rules that paint snow over your platforms, you can simply add the Snow biome value to this group, and it will only show up if the current level uses the Snow enum value.
